Для запуска потребуется python 3.11 и установленные библиотеки: scapy, requests, tkinter<br>
1. Запустите программу с помощью sudo python3.11 main.py
2. В поле ip введите ip адреса через запятую, можно ввести диапазон(пример: 8.8.8.\*[10-20] все адреса начиная с 8.8.8.10, 8.8.8.11, 8.8.8.12, ... , 8.8.8.20)
3. В поле port введите через запятую порты для проверки <br>
Если поле ip не заполнено, то выполняется ввод с файла с тем же синтаксисом. <br>
Если поле port не заполнено, то выполняется скан по всем доступным портам.
